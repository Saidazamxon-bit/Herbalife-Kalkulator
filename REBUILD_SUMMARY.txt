STYLING
├── From: CSS Modules (styles-37aL4.css)

├── Why:  No CSS bloat, dynamic theming, consistency

LANGUAGE
├── From: JavaScript
├── To:   TypeScript
├── Why:  Type safety, better IDE support, fewer runtime errors

DATA MANAGEMENT
├── From: Hardcoded in component
├── To:   Centralized /data/products.ts
├── Why:  Easier updates, reusable, can easily migrate to database

================================================================================
                          TROUBLESHOOTING
================================================================================

App won't start?
$ rm -rf node_modules
$ npm install
$ npm run dev

Styling looks wrong?
• Restart dev server (Ctrl+C then npm run dev)
• Clear browser cache (Ctrl+Shift+Delete)

Images not loading?
• Check paths in /data/products.ts
• Verify files exist in /public/images/
• Check browser console (F12 → Console)

Prices not updating?
• Verify discount selector is working
• Check discount values match product prices
